About the Game:
-
Memory Game is a simple educational game played with a set of cards. Each card displays an Arabic numeral (from 1 to 10), and every number appears on two cards.

The game begins with all cards facing down. The player’s goal is to find matching pairs by flipping two cards at a time. When a correct match is made, the game plays an audio clip of the number pronounced in Arabic, reinforcing learning through visual and auditory cues.
